# Blog Application – Functional Requirements

## 1. Project Overview

The Blog Application allows users to create, manage, and read blog posts.

## 2. Objectives

* Enable users to create blog content
* Provide secure authentication
* Build a scalable web app

## 3. User Roles

### Guest

* View blog posts

### Registered User

* Create posts
* Edit/delete own posts
* Login/logout

## 4. Features

### Authentication

* Register (username, email, password)
* Login
* Logout
* Password hashing

### Blog System

* Create post
* Edit post
* Delete post
* View posts

### Pages

* Home
* Single post
* Login
* Register
* About
* Contact

### Authorization

* Only logged-in users can create posts
* Only owners can edit/delete

## 5. Functional Requirements

* System must support authentication
* System must allow CRUD operations
* System must restrict unauthorized access

## 6. Future Features

* Comments
* Likes
* Profiles
* Admin dashboard
